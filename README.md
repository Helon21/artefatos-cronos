# Artefatos do sistema Cronos

## Para melhor visualização do diagrama abra-o no navegador.

### O diagrama de classes foi elaborado seguindo as classes e relacionamentos existentes do sistema cronos, para que a documentação seja fiel ao que já existe e está em funcionamento. Os requisitos funcionais foram analisados com base no uso do sistema, os não funcionais com base nas tecnologias presentes e na usabilidade, e as regras de negócio foram extraídas de regras definidas dentro do próprio sistema.
